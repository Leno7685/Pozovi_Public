Projekat "Pozovi" predstavlja web sistem za upravljanje događajima, sa različitim nivoima pristupa: gost, registrovani korisnik, zvanica i administrator.
Projekat sadrži i mobilnu aplikaciju koja komunicira sa web sistemom preko API endpoint-a u JSON formatu, pokrivajući sve CRUD funkcionalnosti. Repozitorijum te aplikacije: https://github.com/Leno7685/Pozovi_MobileApp
Korišćene tehnologije i alati: HTML, CSS, JavaScript, PHP, MySQL, Bootstrap, PHPMailer, Composer, DataTables, Git, React Native.
